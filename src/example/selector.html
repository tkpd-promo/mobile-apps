Types

By default there are three types of selector, usage of these selectors depends on the function.

(1) Radio button is used to select a single option from the list, (2) Checkbox is used to select multiple option from the list, (3) Toggle is used to toggle single option.

Guidelines

Padding

(1) Dimension for checkbox and radio button are the same at 20x20px, (2) Selected indicator for radio button are sized 18px, use 2px margin for icons inside toggle/checkbox. (3) Toggle button dimension.

Color

(1) There are two gradient fill colors for selector (2) Usage gradient color in active state (3) Color in inactive state. (4) Hover state use overlay background color of 20% opacity, (5) Background color use same gradient color as in inactive state, only the head part changes color.
